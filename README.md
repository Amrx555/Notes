## ✨ Features
* Organized Layout: A simple interface to manage your daily tasks and notes.
* Fully Responsive: Write and check your notes on any device.
* Clean UI: Minimalist design to help you focus on your content.
* Fast & Lightweight: Optimized for quick loading and smooth interaction.

## 🛠️ Tech Stack
* vue.js
* HTML5 (Structuring your thoughts)
* CSS3 (Modern & clean styling)
* JavaScript (Managing note logic)
* Netlify (Reliable hosting)
